# Introduction

`rsspell` is a fast and efficient spell checker designed for modern development workflows. It supports multiple file formats and provides a simple, command-line interface for checking documents for typos.

## Key Features

- **Fast & Efficient:** Optimized for speed using Rust and the Hunspell-style affix compression system.
- **Markdown & SVG Support:** Specifically handles text within Markdown and SVG files.
- **Extensible:** Easily install additional dictionaries from the CLI.
- **Offline First:** Includes a built-in `en-US` dictionary for immediate, offline use.
